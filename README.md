Yui 一个vue ui组件库，仅做自测研究使用

## 安装
安装组件
``` js
npm i -S vue-yui
or
yarn add vue-yui
```

本组件库使用的是border-box;请设置
``` css
*,*::after,*::before{box-sizing: border-box;}
```
引入
``` js
import {Button,Icon} from 'vue-yui'
import '../node_modules/vue-yui/index.css'
import '../node_modules/vue-yui/dist/index.css'
```
在组件内注册即可使用
